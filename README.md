# Express-GithubAPI
Simple RestAPI using Express that makes calls to GitHubAPI_v3 to fetch relevant data

## Requeriments 
- npm v9.5.1
- node v19.8.1

## Node Dependency
- axios (v1.3.4)
- cors (v2.8.5)
- dotenv (v16.0.3)
- express (v4.18.2)

## Installation
1. Run < npm install >
2. "Optional": Create .env and put your API key

```
API_KEY="YOUR_TOKEN"
```

2. Run < node server.js>
3. Now your server is running on Port 3000!
